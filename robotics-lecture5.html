Pulse Width Modulation

Definitions

  • Digital Voltage - Can be completly on or completely off.
  • Analog Voltage - Can be completly on, completly off or a value in between.
  • Pulse- Is a voltage provided for a given time period.
  • Width- A dimension which we refer to as the the lenght of the graphed pulse.
  • Modulation- Modifying something.

PWM

What does Pulse Width Modulation do?

  • Changes Digital Voltage to Analog Voltage.

Why is this useful?

  • Allows change in motor rotation speed.
    • Without this the motor will only be completly off or completly on.
    • With this the motor will be able to turn at speeds between max ,completly on, and min, completly off.

How does this work?

  • Fluctuate between max voltage and minimum voltage.
    • Turns the voltage on and off quicker than the eye can see, meaning that we will not be able to see it flicker.
    • Instead we will see the average of the small amount of time it is on and the small amount of time it is off.
  • There are two ways of coding

How to use this on arduino.

  • The method available for the ~ marked pins.
  • The method that can be used for any pin.

1. The ~ marked pins method

There is a built in function called analogWrite(pin, value). Where the pin is the integer number of the pin you want to use and the value is a integer number between 0 and 255. The pin has to be marked with a ~ otherwise this will not work. When the value is set to 0, the voltage will be zero all the time (voltage delivered = 0). When the value is 255, the voltage will be on all the time (voltage delivered = Maximum Voltage).

2. 2. Universal Method

This method can be used on any of the Arduino pins. Recall that Pulse Width Modulation (PWM) is basically just a voltage alternating between Vmax and Vmin. Knowing this Arduino allows us to manualy make our own PWM pin by switching the pin on for a few micro seconds and back off for a few micro seconds. Meaning that the ration between the time on and time of will be the voltage delivered. This method have the advantage that we can use any output pin and that there is more control, but there is slightly more mathematics.
