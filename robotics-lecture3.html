Arduino Logic

Arduino logic is how we allow computers to make desicions. It checks to see if a certain definition is true, if it is true it will run certain code and if it is false it will run different code.

Let us look at the following example. Let x eqaul to 3, John's grade. Now I tell you; if x is bigger than 5 you will pass the course, else John failed. From the example you can see that John failed. Would it not be nice if your computer would automatically know hweter or not John failed? There is a way to do this.

It is called logic.

List of comparison operators:

  • == Is it eqaul to?
  • != Is it not eqaul to?
  • >= Is it bigger or eqaul to?
  • <= Is it smaller or eqaul to?

  • --
